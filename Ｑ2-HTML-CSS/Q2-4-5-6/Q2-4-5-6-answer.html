5. Please explain about what is CSS boxing model and the layout components that it consists of.說明盒子模型

CSS Box Model描述一個網頁元素如何被渲染在網頁上。

盒子模型的組件：

  • Content（內容區）：實際的內容區域
  • Padding（內距）：內容與邊框之間的空間
  • Border（邊框）：包圍元素的邊框
  • Margin（外距）：元素與其他元素之間的空間

box-sizing 屬性：

  • content-box（預設值）：
    • width/height 只包含內容區域的尺寸
    • padding 和 border 會額外加到元素總尺寸上
  • border-box：
    • width/height 包含內容區域、padding 和 border
    • 設定的尺寸就是元素最終的尺寸
    • 因為較為直觀，常見的 CSS reset 會設定： 
      * { box-sizing: border-box; }
